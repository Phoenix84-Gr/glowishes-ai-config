# Glowishes Instagram Skill

## Store context
Glowishes (@glowishes_com) — Greece-based K-beauty Shopify store. ~350 followers. Growing phase. Primary goal: reach, saves, shares → followers → sales.

## Algorithm priorities (2025-2026)
1. **Reels** — highest reach multiplier. Priority format.
2. **Carousels** — highest saves + shares. Best for education.
3. **Stories** — engagement (polls, quizzes) + retention.
4. **Static posts** — lowest reach. Use only for announcements.

## What the algorithm rewards
- Watch time (Reels): hooks that stop scroll in 0-1.5s
- Saves: "save this for later" content (routines, tips, comparisons)
- Shares: relatable, surprising, or educational content
- Comments: posts that ask questions or spark debate
- Profile visits after Reel → follow conversion

## What kills reach
- Low watch time (boring hook)
- No CTA
- Posting and ghosting (no replies to comments in first 30 min)
- Overly promotional (push product without value)
- Bad aspect ratio (always 9:16 for Reels, 4:5 for feed, 1:1 for square)

## Content formats

### Reels (priority)
```
Duration: 7-30 seconds (sweet spot: 10-18 seconds)
Aspect ratio: 9:16 (1080x1920)
Hook: 0-1.5s → visual interrupt + text overlay
Structure:
  0.0s: Pattern interrupt (weird angle, bold text, movement)
  1.5s: Problem or promise ("If you do X you need to see this")
  5s:   Product/routine in action
  12s:  Result or key benefit
  16s:  CTA (text overlay + voiceover)
Audio: trending sound preferred. Use CapCut trending library.
Text overlay: always. Makes it watchable on mute.
```

### Carousel (saves machine)
```
Slide count: 5-10 slides
Slide 1: Hook — must make them swipe
Slide 2: Problem / common mistake
Slide 3-7: Value (tips, steps, facts, comparison)
Last slide: CTA to save + follow
Caption: short, adds context, ends with question
```

### Stories (daily engagement)
```
Types that work:
- Poll: "Which sunscreen texture do you prefer?"
- Quiz: "What does SPF 50 protect against?"
- Slider: "Rate your skincare routine"
- Behind the scenes: packing orders, new arrivals
- Product demo: fast 15-second clip
Frequency: 3-7 stories per day
```

## Caption structure
```
Line 1: Hook (repeats or extends the video hook)
[blank line]
Value lines (2-5 lines, short)
[blank line]
CTA: Question OR instruction ("Save this", "Tag a friend who needs this")
[blank line]
Hashtags: 3-8 targeted hashtags (not 30)
```

## Hashtag strategy
Mix:
- 1-2 large (1M+): #kbeauty #skincare
- 2-3 medium (100K-1M): #kbeautyroutine #sunscreengreece
- 2-3 small (10K-100K): #glowishes #kbeautygreece #koreanskincaregr
- 1 Greek: #ελληνική_ομορφιά or niche

## Posting schedule (Greece timezone EEST/EET)
Best times:
- Weekdays: 7:00-9:00pm
- Weekends: 11:00am-1:00pm, 7:00-9:00pm
Frequency: 4-5 Reels/week + daily Stories

## Greek language rules
- Default: Greek
- Product names: keep in English/Korean (Beauty of Joseon, Anua, etc.)
- Tone: friendly, direct, no medical claims
- Slang: modern Greek (OK to use "glow", "K-beauty" as loanwords)

## Required outputs per request
1. Hook (text + visual direction)
2. Platform format (Reel / Carousel / Story)
3. Slide-by-slide or scene-by-scene breakdown
4. Caption (Greek)
5. Hashtags
6. Audio suggestion (vibe/genre if no specific track)
7. CTA
8. Compliance check (no medical claims)

## Compliance rules
Never use:
- "θεραπεύει ακμή" (treats acne)
- "αφαιρεί ρυτίδες" (removes wrinkles)
- "ιατρική επίδραση" (medical effect)
- "εγγυημένα αποτελέσματα" (guaranteed results)
- "δερματολογικά αποδεδειγμένο" unless you have the cert

Prefer:
- "βοηθά να φαίνεται το δέρμα πιο..." (helps skin look more...)
- "κατάλληλο για δέρμα που..." (suitable for skin that...)
- "ελαφριά αίσθηση" (lightweight feel)
